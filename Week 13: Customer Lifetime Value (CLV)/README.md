# README — Customer Lifetime Value (CLV) and Retention Analysis

## Overview

This week introduces **Customer Lifetime Value (CLV)** and **customer retention analysis**, two fundamental concepts in marketing analytics. Firms use CLV to estimate the long-term value generated by each customer and to guide decisions about customer acquisition, retention, and marketing investment.

In this exercise, students work with real transaction-level retail data to compute customer-level metrics and estimate simple CLV measures. The analysis also introduces a basic retention model to understand which customers are more likely to continue purchasing.

Students will learn how to:

- Calculate customer-level metrics from transaction data
- Estimate historical Customer Lifetime Value (CLV)
- Build a simple retention model
- Identify high-value and at-risk customers
- Translate analytical results into marketing strategy insights


## Dataset

This exercise uses the **Online Retail dataset**, which contains transactional data from a UK-based online retailer between **December 2010 and December 2011**.

Each row represents a transaction line item from an invoice. Customers may appear multiple times in the dataset because they make multiple purchases over time. This structure allows us to compute metrics such as purchase frequency, total revenue per customer, and recency.

The dataset contains over **500,000 transaction records**, making it suitable for realistic marketing analytics exercises involving customer value and retention.


## Key Variables

ariable: Description

InvoiceNo: Unique invoice number

StockCode: Product code

Description: Product description

Quantity: Number of units purchased

InvoiceDate: Date and time of the transaction

UnitPrice: Price per unit

CustomerID: Unique customer identifier

Country: Country where the customer reside


Additional variables are created during the analysis:

Derived Variable: Meaning

revenue: Transaction revenue (Quantity × UnitPrice)

frequency: Number of purchases made by a customer

avg_order_value: Average order value

recency_days: Days since the customer's last purchase

historical_clv: Total revenue generated by the customer

## Use in This Course

In this week's materials, the dataset is used to demonstrate:

- Cleaning and preparing transaction-level data
- Aggregating transactions into customer-level metrics
- Calculating historical CLV
- Building a simple retention model using logistic regression
- Segmenting customers based on their lifetime value

Students will also explore how CLV insights can inform marketing strategies such as retention campaigns, loyalty programs, and targeted promotions.
